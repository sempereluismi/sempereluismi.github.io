¿Qué es tailwind CSS?

Tailwind CSS es un framework CSS que permite un desarrollo ágil, basado en clases de utilidad que se pueden aplicar con facilidad en el código HTML y unos flujos de desarrollo que permiten optimizar mucho el peso del código CSS.

Como instalar tailwind CSS

Hay dos formas de utilizar tailwind CSS, Estas son con el link de un cdn o utilizando nodejs con npm

Con CDN

Insertamos el siguiente código

code

<head>

<script src="https://cdn.tailwindcss.com></script>

</head>

Usando NPM

Ejecutamos los siguientes comandos

terminal

npm install -D tailwindcss

npx tailwindcss init

Editamos tailwind.config.js

module.exports = {

content: ["./src/**/*.{html,js}"],

theme: {

extend: {},

},

}

Creamos un CSS y insertamos

@tailwind base;

@tailwind components;

@tailwind utilities;

ejecutamos este comando

npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
